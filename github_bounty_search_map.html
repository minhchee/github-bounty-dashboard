研判 三个该正视的事实

红皇后效应
  • bounty 总数万级，但被无数自动 agent（BountyScout 等）实时扫食
  • 「未认领」窗口以分钟计，索引延迟会制造虚假空池
  • 对策：认领前实时 GET 校验 comments，不依赖搜索快照
不对称风险收益
  • Algora 真实 USD + 有 completed 记录 = 确定性高
  • Aigen 代币 + 失活创始人 = 确定性低（已证伪）
  • RTC/crypto token = 近乎零价值
KYC 矛盾
  • 记忆：Algora 需 Stripe+KYC
  • 外网 2026 文：no KYC, just GitHub
  • 必须实测，不能假设任一方为真
我的现实锚点
  • 已握 10 个 hiragana PR（#6989–6998）等合入 ≈$500
  • 新产出应转向真实 USD 流，而非再投饱和微任务
